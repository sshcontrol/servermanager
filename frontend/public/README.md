# Public assets

Place **bg.png** here to use a custom full-screen background on the login page.

- File name: `bg.png`
- Recommended: high resolution (e.g. 1920×1080 or larger), can be subtle so the login card stays readable.

If bg.png is not present, the login page uses a Persian-style gradient background.

## Open Graph image (social sharing)

Place **poster.jpg** (or any image) here to use as the Open Graph image when sharing links on social media.

- File name: `poster.jpg` → URL: `https://yourdomain.com/poster.jpg`
- Recommended: 1200×630px for best display on Facebook, Twitter, etc.
- Set the URL in **Superadmin → Settings → SEO → Open Graph Image URL**.
